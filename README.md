# Login Form UI (HTML & CSS)

## 📌 Objective
This project is a basic login form interface built using HTML and CSS.  
The goal is to practice form structure, layout, and UI styling.

---

## 🚀 Features
- Login form layout with email/phone and password fields
- Required field validation using native HTML attributes
- Styled input fields with icons
- Submit button and navigation links (forgot password, signup)

---

## 🛠 Technologies Used
- HTML5
- CSS3
- Font Awesome (icons)
- Google Fonts (Poppins)

---

## 📚 What I Learned
- Structuring forms using HTML
- Using input attributes like `required`
- Styling form elements with CSS
- Positioning icons inside input fields
- Creating basic UI layouts with Flexbox

---

## ⚠️ Limitations
- No JavaScript validation or logic
- Form does not submit data to a backend
- No real authentication functionality
- No responsiveness implemented with media queries
- No accessibility improvements (labels, ARIA, etc.)

---

## 🔧 Future Improvements
- Add JavaScript validation
- Connect form to a backend (authentication system)
- Implement proper responsiveness with media queries
- Improve accessibility with labels and ARIA attributes
- Add error messages and user feedback
